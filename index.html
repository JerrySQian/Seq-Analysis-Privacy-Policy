Privacy policy

We Collect No Personal Information

We do not collect any personal information. All sequences inputted are saved locally. In addition, the current version of Seq Analysis only stores the 10 most recent sequences inputted. The NLimit value and stability results are likewise saved locally. Data that is saved locally is shown directly in the app, so in the case that users wish to access what data is saved, it is possible.

Users do not have the option to input personal, sensitive, or proprietary information, thereby ensuring complete privacy. Attempts to input personal information and save it locally through Seq Analysis is not possible due to rigorous input processing.
